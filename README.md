## 🚀 Instalando `<dashgo>`

Para instalar o `<dashgo>`, siga estas etapas:

Clone o projeto:
```
<git clone https://github.com/danielafarias/dashgo.git>
```
Em seguida, adicione as bibliotecas e dependências necessárias:
```
<cd dashgo>
```
```
<yarn>
```

## 🧑‍💻 Usando `<dashgo>`

Para usar `<dashgo>`:

```
<yarn dev>
```
